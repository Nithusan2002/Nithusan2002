## 🚀 LiftOff

🔗 [github.com/Nithusan2002/LiftOff](https://github.com/Nithusan2002/LiftOff)

Android app that evaluates rocket launch conditions based on weather data for a given location and time.

The non-trivial part: Meteorologisk institutt's IsoBaricGrib API returns binary GRIB files — a format Android can't parse directly. We built a custom backend server on Heroku to handle the binary parsing and expose the data as a clean REST API the app could consume.

**Stack:** Kotlin · Jetpack Compose · MVVM · REST API · Heroku
